## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/skilldent-clinic.git
   cd skilldent-clinic
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
